真实开发环境使用webpack打包工具，帮助构建大型项目开发目录，在开发完成之后进行打包的操作。

自己配置webpack难度较大，vue 官方提供一款脚手架工具即 vue-cli

<快速构建标准的 vue 项目，同时自带webpack各种配置，可以迅速上手工程级别项目的开发。技术门槛大幅降低，快速进行开发。

vue-cli 的安装及使用
官网 >> 学习 >> 教程 >> 安装 >> 命令行工具 (CLI) >> Vue CLI 的文档

CLI 会被全局安装并在终端里提供 vue 命令：

npm install -g @vue/cli

// 报错 ：Please try running this command again as root/Administrator.

// 改用 管理员 模式 sudo npm install -g @vue/cli 运行

vue create my-project

//vue-cli / my-project 文件代码

// 可以选择淘宝镜像 Use https://registry.npm.taobao.org for faster installation? No

// Please pick a preset: default (babel, eslint)

//cd my-project

//App running at:

//- Local: http://localhost:8080/

//- Network: http://192.168.1.104:8080/

主要文件

主要在 src 文件里编写文件

main.js 整个项目的入口文件。挂载点：public 中的 ‘<div id="app"></div>’引入组件使用 import App from './App.vue';

app.vue 中三个部分： template、javascript、style；单文件组件，每个vue文件包含了一个组件所需要的所有内容。

文件更新保存好后不需要刷新网页 及为所得，项目内值webpack 的 webpack-dev-server 自动刷新

可以大胆的使用 ES6 的语法，脚手架会将 ES6 语法 打包成 ES5 语法；
